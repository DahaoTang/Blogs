# Markdown Syntax Guide

This document showcases **all supported Markdown grammar** commonly used in GitHub READMEs.

```markdown

### Heading 3

#### Heading 4

##### Heading 5

###### Heading 6
```

# Heading 1

## Heading 2

### Heading 3

#### Heading 4

##### Heading 5

## 2. Text Formatting

### Bold and Italic

```markdown
**Bold**
_Italic_
_Italic_
**_Bold and Italic_**
```

**Bold**  
_Italic_  
_Italic_  
**_Bold and Italic_**

---

## 3. Lists

### Unordered List

```markdown
- Item 1
  - Sub-item 1
  - Sub-item 2
- Item 2
```

- Item 1
  - Sub-item 1
  - Sub-item 2
- Item 2

---

### Ordered List

```markdown
1. Item 1
2. Item 2
   1. Sub-item 1
   2. Sub-item 2
```

1. Item 1
2. Item 2
   1. Sub-item 1
   2. Sub-item 2

---
